# TextViewExpand
能减少布局的嵌套只要一个控件

## 引用方法（gradle）
Add it in your root build.gradle at the end of repositories:


    allprojects {
		  repositories {
			  ...
			  maven { url 'https://jitpack.io' }
		  }
	  }
    
Add the dependency
  
    dependencies {
	        compile 'com.github.litterLeaf:TextViewExpand:1.0.0'
	  }
